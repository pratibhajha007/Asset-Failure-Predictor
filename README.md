# ⚙️ Asset Failure Prediction using Machine Learning

## 📌 Overview

This project predicts whether a machine is likely to fail based on key operational parameters such as temperature, rotational speed, torque, and tool wear. The goal is to demonstrate the concept of predictive maintenance, helping prevent unexpected breakdowns and improve efficiency.

---

## 🧠 Problem Statement

Machines in industries often fail unexpectedly, leading to downtime and financial loss. This project aims to predict machine failure in advance using machine learning so that maintenance can be performed proactively.

---

## ⚙️ Features

* Predicts machine failure (0 = Safe, 1 = Failure)
* Uses Random Forest algorithm
* Handles real-world inspired sensor data
* Provides quick and accurate predictions

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn

---

## 📊 Dataset

A structured dataset containing:

* Air temperature
* Process temperature
* Rotational speed
* Torque
* Tool wear
* Machine failure (target)
